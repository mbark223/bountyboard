# Deployment Configuration

This project is deployed on Vercel with the following configuration:

## Environment Variables
- `DATABASE_URL` - PostgreSQL connection string for Supabase

## Node.js Version
- Using Node.js 20.x

## API Endpoints
- `/api/briefs` - Get published briefs
- `/api/admin/briefs` - Get all briefs (admin)
- `/api/admin/submissions` - Get submissions for a brief
- `/api/test-simple` - Test endpoint to verify deployment
